# serpent

## Description
Un jeu de serpent en python avec tkinter crée dans le context de projet de fin d'annes en Bac S SVT spécialité ISN.
## Fonctionnalités
- Du déplacement
- D’allongement
- De directions du serpent
- De création :
  - de corps du serpent 
  - et les pommes aléatoires, 
  - de fenêtre primaire et des fenêtres secondaire (règles du jeu + manuel d'utilisation + fin de partie)
## PreRequis - Technologie
- Python 3
- pip3
  - tkinter (GUI)
  - random
